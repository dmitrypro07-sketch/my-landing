## Технические детали index.html

- **Размер:** ~1250 строк, всё в одном файле
- **Библиотеки:** только Google Fonts (Golos Text + Inter + JetBrains Mono)
- **JS:** ~200 строк Vanilla JS
  - `IntersectionObserver` для reveal-анимаций
  - `IntersectionObserver` для активного пункта nav
  - `animateCounter()` для цифр с easeOut
  - `runChatDemo()` — цикличная анимация диалога
  - FAQ accordion через CSS `grid-template-rows`
  - Exit-intent через `mouseleave` event (только desktop)
  - Mobile sticky bar через scroll position (после 28% страницы)
  - Динамический месяц в CTA (следующий от текущей даты)
  - Валидация email в exit-попапе
- **Структура CSS:** custom properties → reset → typography → layout → components → каждая секция → responsive
- **SEO:** title, description, OG-теги, JSON-LD FAQ, canonical
